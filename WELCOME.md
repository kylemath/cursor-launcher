## Add New Projects

To add a project to the dashboard:

1. Create `catalogue.json` in project root:
```json
{
  "id": "my-project",
  "title": "My Project",
  "oneLiner": "What it does",
  "tags": ["tag1", "tag2"]
}
```

2. Add `screenshot.png` 

3. Run: `projects`
